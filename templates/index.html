About this project

Stack overflow is a popular Q&A site for programming engineers. Expert in different domains help others out. Using machine learning algorithms and the big data tools, we analyzed 20G stack overflow data. We classify among all the users on all the tags. We find expert similarities and provide contact informations on popular experts.

Why useful?

Stack overflow user reputation doesn't reflect in which domain a user is good. From the data of all the users' behavior on all the tags, experts are identified by machine learning classification algorithm.
